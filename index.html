Resident Privileges

แยกการใช้งานเป็น 2 path แล้ว: Resident path และ Admin path

Resident path สำหรับแขก/เจ้าของห้อง ใช้เมนูด้านล่าง 4 ปุ่ม: Home, Redemption, Member, Settings ส่วน Admin path ใช้สำหรับทีมโรงแรมและ CMS หลังบ้าน

Resident Loginหน้า login แยกสำหรับลูกค้า Resident Resident Homeบัตรสมาชิก คะแนน และสรุปสิทธิ์ Newsข่าวสาร Resident แยกหน้า Promotionsโปรโมชันแยกหน้า Benefitsสิทธิประโยชน์แยกหน้า Admin Dashboardบันทึก spend และ dashboard Membersค้นหาสมาชิกแยกหน้า
